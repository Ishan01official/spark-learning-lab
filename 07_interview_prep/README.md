# 07 — Interview prep & Databricks certification

Focused prep for two related goals:
1. **Pass the Databricks Certified Associate Developer for Apache Spark Using Python.**
2. **Walk into a Spark / PySpark technical interview confident.**

The notes here are condensed reference material. The two mock exams at the bottom are calibrated to the cert's actual difficulty.

## What you should be able to do after this

- Sit the Databricks Associate Developer exam with a clear strategy.
- Answer code-reading questions about Spark plans, repartitioning, joins, and streaming.
- Reproduce the common PySpark patterns from memory.
- Explain trade-offs between AQE, broadcast, salting, caching.

## How to use this module

Two weeks before the exam:
- Day 1–3: review cheatsheet + code patterns. Run every example.
- Day 4–7: work through mock exam 1, review every wrong answer.
- Day 8–11: work through mock exam 2, focus on weak topics.
- Day 12–13: re-read pitfalls; review optimization (module 03) and streaming (module 05).
- Day 14: light review, get a good night's sleep.
