# 08 — Chapter-by-chapter book notes

Distilled notes from the four O'Reilly references that drove this curriculum. Each note is a one-page summary you can re-read in 5 minutes to refresh on what a chapter taught.

These aren't replacements for the books — they're memory aids. Read the source for depth; use these to find your place again.

## Books covered

1. [Learning Spark, 2e](learning-spark-2e/) — Damji, Wenig, Das, Lee (2020)
2. [High Performance Spark, 2e](high-performance-spark-2e/) — Karau, Warren (2024)
3. [Data Algorithms with Spark](data-algorithms-with-spark/) — Parsian (2022)
4. [Delta Lake: The Definitive Guide](delta-lake-definitive-guide/) — Armbrust, Das, Powers, Tomar (2024)

## How these notes are organized

Each chapter gets a short note containing:
- **Key concepts** (~5 bullets) — what the chapter is *about*.
- **Most useful APIs / code** — small code snippets to remember.
- **Best examples in the chapter** — page-numbered pointers if you have the book.
- **Connection to other modules in this lab** — where else we cover the same ground.
- **One-line takeaway** — the single sentence to walk away with.

## Suggested reading order

If you're just starting:
1. *Learning Spark, 2e* — Chapters 1–5 (foundations).
2. *Learning Spark, 2e* — Chapters 6–9 (DataFrames, SQL, optimization).
3. *High Performance Spark, 2e* — pick chapters that match your bottleneck.
4. *Delta Lake: The Definitive Guide* — Chapters 1–8 for Delta basics.
5. *Data Algorithms with Spark* — when you need patterns for specific problems.
