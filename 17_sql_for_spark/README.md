# 17_sql_for_spark - SQL Basics Needed For Spark SQL

Spark SQL is one of the most important Spark interfaces. This module covers the SQL required for the rest of the repo.

## Topics

- `SELECT`
- `WHERE`
- `GROUP BY`
- `HAVING`
- joins
- CTEs
- window functions
- null handling
- dates and timestamps
- arrays, structs, maps
- temp views
- query plans

## Learning Objectives

- Translate basic DataFrame code into SQL.
- Read SQL query plans in Spark UI.
- Avoid common null and join mistakes.
- Use windows for ranking and deduplication.

## Practice Loop

1. Write the SQL.
2. Write the equivalent DataFrame code.
3. Run both.
4. Compare plans.
5. Explain which is easier to maintain.
