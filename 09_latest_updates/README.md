# 09 — Latest Spark updates and the road ahead

What's changed in Spark recently, what's coming, and what it means for how you write code.

This module focuses on the *deltas* between what's in *Learning Spark, 2e* (2020) and current Spark. If you're studying for the cert from older material, this module covers the gap.

## What you should be able to do after this

- Know what's new in Spark 3.5 and what's coming in 4.0.
- Understand Spark Connect and why it matters.
- Make a sensible choice between Spark-on-Kubernetes, EMR, Databricks, and Glue.
- Know the relevance of Photon and other native execution engines.
